# Projeto Tela de Login 🔐

Este é um projeto simples e funcional de uma página de login, desenvolvido para praticar a integração entre HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- Interface responsiva e moderna.
- Validação de dados via JavaScript.
- Sistema de alerta para o usuário:
  - **Sucesso:** Ao inserir as credenciais corretas, o sistema exibe a mensagem: `!Login realizado com sucesso!`.
  - **Erro:** Se o nome ou a senha estiverem incorretos, o sistema exibe: `Nome ou senha incorretos!`.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação da página e dos campos de entrada.
* **CSS3:** Estilização visual, cores e layout.
* **JavaScript:** Lógica de validação das credenciais e manipulação do DOM.
